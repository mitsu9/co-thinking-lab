# アイデア発想・ブレストスキル

創造的にアイデアを発散し、新しい可能性を探るための思考セッションを開始します。

## 実行手順

### 1. 基盤の読み込み
- `knowledge/constitution.md`
- `knowledge/personas/` (3ファイル)
- `knowledge/frameworks/brainstorming.md`
- `templates/brainstorming.md`

### 2. ブレストの実施
1. **テーマと目的の設定**: 何についてアイデアを出すのか
2. **発散フェーズ**: 批判禁止で量を重視し、多くのアイデアを出す
3. **収束フェーズ**: アイデアをグルーピングし、評価基準で選別する
4. **精緻化**: 選んだアイデアを具体化する

### 3. 思考ツールの活用
マインドマップ、SCAMPER法、ランダムワード、What if...?などを活用

### 4. ペルソナからのフィードバック
各ペルソナの視点から、有望なアイデアを評価

### 5. セッションの保存
`sessions/YYYY/YYYY-MM-DD_brainstorming_[トピック名].md` に保存

## ブレストの4つのルール
1. 批判禁止 2. 自由奔放 3. 量を重視 4. 結合改善

**創造的で実践的なアイデアを生み出すことを支援します。**
