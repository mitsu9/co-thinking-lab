# 思考の可視化・レポート生成スキル

思考プロセスを可視化し、レポートを生成します。

## 実行手順

### 1. レポートの対象を確認
ユーザーに、レポート化したい対象を確認します:
- 単一の思考セッション
- 複数のセッションの統合分析
- 特定期間の思考パターン分析

### 2. データの収集と分析
指定された対象のファイルを読み込み、分析します:
- 使用した思考パターンの頻度
- 主な決定事項とアクション
- ペルソナからのフィードバックの傾向
- 繰り返し現れるテーマや課題

### 3. レポートの生成
マークダウン形式でレポートを生成します:

```markdown
# 思考プロセスレポート

**対象期間**: YYYY-MM-DD 〜 YYYY-MM-DD
**生成日**: YYYY-MM-DD

## サマリー
[期間中の思考活動の概要]

## 思考パターンの分析
| 思考パターン | 回数 | 割合 |
|--------------|------|------|
| 問題分析 | X | XX% |
| 意思決定 | X | XX% |
| ブレスト | X | XX% |

## 主なテーマ
1. [テーマ1] - [頻度]
2. [テーマ2] - [頻度]
3. [テーマ3] - [頻度]

## 主な決定事項
- [決定1]
- [決定2]
- [決定3]

## アクションアイテムの進捗
[実行されたアクション、未実行のアクション]

## ペルソナからのフィードバックの傾向
- 事業責任者: [主な懸念事項]
- チームメンバー: [主な懸念事項]
- エンドユーザー: [主な懸念事項]

## 学びと洞察
[期間を通じて得られた主な学び]

## 推奨事項
[今後の思考活動への提案]
```

### 4. 可視化（オプション）
必要に応じて、思考パターンの分布や傾向をテキストベースのチャートで表現します。

**思考の軌跡を可視化し、継続的な改善を支援します。**
