# 過去の思考の検索スキル

キーワードや日付で過去の思考ログを検索します。

## 実行手順

### 1. 検索条件の確認
ユーザーに検索条件を確認します:
- キーワード: 検索したいキーワード
- 思考パターン: problem-analysis, decision-making など
- 期間: 日付範囲

### 2. 検索の実行
`sessions/` ディレクトリ内を検索します。

#### ファイル名での検索
Glob ツールを使用して、ファイル名パターンでマッチング:
- 特定の思考パターン: `sessions/**/*decision-making*.md`
- 特定の日付範囲: `sessions/2026/2026-02-*.md`

#### 内容での検索
Grep ツールを使用して、ファイル内容を検索:
- キーワード: `pattern: "キーワード", path: "sessions/"`

### 3. 検索結果の整理
マッチしたファイルのリストと、簡潔な要約を提供します:

```markdown
## 検索結果

### [ファイル名1]
- **日付**: YYYY-MM-DD
- **思考パターン**: [パターン]
- **概要**: [簡潔な説明]

### [ファイル名2]
- **日付**: YYYY-MM-DD
- **思考パターン**: [パターン]
- **概要**: [簡潔な説明]
```

### 4. 詳細表示（オプション）
ユーザーが興味を持ったファイルについて、詳細を表示できます。

**過去の思考を効率的に検索し、再利用することを支援します。**
