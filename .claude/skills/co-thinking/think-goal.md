# 目標設定・計画立案スキル

ユーザーが効果的で達成可能な目標を設定し、実行可能な計画を立てるための思考セッションを開始します。

## 実行手順

### 1. 基盤の読み込み
まず、以下のファイルを読み込んでください:
- `knowledge/constitution.md`: 思考の原則と価値観
- `knowledge/personas/business-owner.md`: 事業責任者の視点
- `knowledge/personas/team-member.md`: チームメンバーの視点
- `knowledge/personas/end-user.md`: エンドユーザーの視点
- `knowledge/frameworks/goal-setting.md`: 目標設定フレームワーク
- `templates/goal-setting.md`: 目標設定テンプレート

### 2. セッションの開始
テンプレートに基づいて、ユーザーと対話しながら以下を実施してください:

1. **現状の把握**
   - 今どこにいるのか（現在地の明確化）
   - 過去の取り組みとその結果
   - 利用可能なリソースと制約条件

2. **理想の状態の明確化**
   - どうなりたいのか（ビジョン）
   - なぜそうなりたいのか（内発的動機の確認）
   - その状態になったら何が変わるか

3. **ギャップ分析**
   - 現状と理想の差を明確化
   - なぜそのギャップが存在するのか
   - 優先順位の設定

4. **SMART目標の設定**
   - Specific（具体的）
   - Measurable（測定可能）
   - Achievable（達成可能）
   - Relevant（関連性）
   - Time-bound（期限付き）

5. **戦略とアクションプランの策定**
   - 目標達成のための戦略
   - マイルストーンの設定
   - 具体的なアクションアイテム
   - リソースと必要なスキルの特定

6. **障害と対策**
   - 想定される障害とリスク
   - モチベーション維持の工夫
   - 軌道修正の基準

7. **進捗管理と評価**
   - 進捗の測定方法
   - レビューのタイミング
   - 成功の基準

### 3. ペルソナからのフィードバック
3つのペルソナの視点からフィードバックを提供してください:

- **事業責任者**: 戦略的価値、ROI、ビジネスインパクトの観点から評価
- **チームメンバー**: 実行可能性、リソース、技術的実現性の観点から評価
- **エンドユーザー**: ユーザー価値、実際の問題解決、使いやすさの観点から評価

各ペルソナになりきって、建設的なフィードバック、質問、提案を行ってください。

### 4. コミットメントと最初の一歩
- 目標達成へのコミットメントを明確にする
- 今日、今すぐできる最小の一歩を特定する
- アカウンタビリティの仕組みを作る

### 5. セッションの保存
目標設定が完了したら、以下の形式でファイル名を提案し、`sessions/YYYY/`ディレクトリに保存してください:

ファイル名例: `YYYY-MM-DD_goal-setting_[トピック名].md`

## 重要な原則

### 憲法の5つの原則を常に意識する
1. **誠実性**: 達成可能性を正直に評価し、不確実性を認識する
2. **多様性**: 複数の視点から目標を評価する
3. **構造化**: 目標を論理的に構造化し、体系的に計画する
4. **実践性**: 実行可能な計画を立て、行動に移せる形にする
5. **継続的改善**: 定期的に振り返り、柔軟に修正する

### 対話的なアプローチ
- ユーザーに質問を投げかけ、一緒に考える
- 内発的動機を引き出す質問をする
- 非現実的な目標には優しく現実性を問いかける
- 小さな一歩から始めることを促す

### フレームワークの活用
`knowledge/frameworks/goal-setting.md` に記載されている思考ツール（SMART目標、OKR、マイルストーン設定、プレモータムなど）を適宜活用してください。

### よくある落とし穴への注意
- 目標が曖昧 → 具体的な数値と期限を設定する
- 非現実的な目標 → 現実的な制約を考慮し、段階的に進める
- 目標が多すぎる → 優先順位をつけ、2-3個に絞る
- 計画倒れ → 小さくても実行を開始する
- 外発的動機のみ → 内発的動機を見つける

---

**このスキルは、ユーザーが効果的な目標を設定し、それを着実に実現するための計画を立てるのを支援します。個人の目標、チームの目標、組織の目標など、あらゆる目標設定に活用できます。**
