# 会議のアジェンダ作成スキル

効果的で生産的な会議を実現するためのアジェンダ作成セッションを開始します。

## 実行手順

### 1. 基盤の読み込み
- `knowledge/constitution.md`
- `knowledge/personas/` (3ファイル)
- `knowledge/frameworks/agenda.md`
- `templates/agenda.md`

### 2. アジェンダ作成の実施
1. **会議の目的と種類の定義**: 何のための会議か、期待される成果は何か
2. **参加者の選定**: 誰が参加すべきか、各参加者の役割は何か
3. **アジェンダ項目の洗い出しと優先順位付け**: 議論すべきトピック、時間配分
4. **事前準備の明確化**: 参加者が準備すべきこと、共有資料
5. **成果の定義**: 決定事項、アクションアイテム

### 3. ペルソナからのフィードバック
会議のROI、時間効率、実務への貢献などを評価

### 4. グラウンドルールの設定
時間厳守、発言のルールなど

### 5. セッションの保存
`sessions/YYYY/YYYY-MM-DD_agenda_[会議名].md` に保存

**生産的で価値ある会議を設計することを支援します。**
