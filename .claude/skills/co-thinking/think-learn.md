# 学習・知識整理スキル

新しい知識を効果的に学び、既存の知識と統合して活用可能にするための思考セッションを開始します。

## 実行手順

### 1. 基盤の読み込み
- `knowledge/constitution.md`
- `knowledge/personas/` (3ファイル)
- `knowledge/frameworks/learning.md`
- `templates/learning.md`

### 2. 学習セッションの実施
1. **学習の目的設定**: なぜ学ぶのか、どこで活用するのか
2. **既存知識の確認**: すでに知っていること、理解できていないこと
3. **新しい知識の獲得**: 情報源から学び、キーポイントを整理
4. **理解の深化**: 自分の言葉で説明し、具体例を挙げる
5. **知識の統合**: 既存知識との関連を明確にする
6. **実践と応用**: 実際に使ってみて、フィードバックから学ぶ

### 3. 学習ツールの活用
ファインマンテクニック、コーネルメソッド、マインドマップなどを活用

### 4. ペルソナからのフィードバック
各視点から、学んだ知識の応用可能性を評価

### 5. セッションの保存
`sessions/YYYY/YYYY-MM-DD_learning_[トピック名].md` に保存

**効果的に学び、知識を活用できるよう支援します。**
